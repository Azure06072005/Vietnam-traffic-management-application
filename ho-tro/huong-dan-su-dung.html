Menu chính

Thanh menu chính của VNTraffic bao gồm các mục:

  • Trang chủ: Quay về trang chính của ứng dụng
  • Tin tức: Thông tin về tình hình giao thông và các bản tin mới nhất
  • Tra cứu phương tiện: Tra cứu thông tin về các phương tiện giao thông
  • Diễn đàn: Diễn đàn thảo luận về các vấn đề giao thông
  • Đấu giá biển số: Thông tin và tính năng liên quan đến đấu giá biển số xe
  • An toàn giao thông: Thông tin và hướng dẫn về an toàn giao thông
  • Hỗ trợ: Trợ giúp, hướng dẫn sử dụng và các thông tin liên hệ

Quản lý dữ liệu

Phần Quản lý dữ liệu cho phép bạn tải lên và quản lý các file dữ liệu để phân tích.

Thêm dữ liệu

Nhấn vào nút "Thêm dữ liệu" (màu xanh lá) để thêm dữ liệu mới vào hệ thống. VNTraffic yêu cầu hai loại dữ liệu:

  1. File GeoJSON: Chứa dữ liệu địa lý của các tỉnh/thành phố Việt Nam.
    • Kéo thả file vào khu vực quy định hoặc nhấn "Chọn file GeoJSON" để tải lên
    • Sau khi tải lên thành công, bạn sẽ thấy thông báo "Đã tải lên: diaphantinh.geojson"
  2. File CSV dữ liệu giao thông: Chứa dữ liệu về vi phạm, tai nạn, và các thông tin liên quan.
    • Kéo thả file vào khu vực quy định hoặc nhấn "Chọn file CSV" để tải lên
    • Sau khi tải lên thành công, bạn sẽ thấy thông báo "Đã tải lên: vipham_updated.csv"

Chỉnh sửa dữ liệu

Nhấn vào nút "Chỉnh sửa dữ liệu" (màu xanh dương) để sửa đổi dữ liệu đã tải lên. Tính năng này cho phép bạn:

  • Cập nhật thông tin trong file đã tải lên
  • Loại bỏ dữ liệu không chính xác
  • Thêm thông tin bổ sung

Cấu hình trực quan hóa

Phần Cấu hình trực quan hóa cho phép bạn tùy chỉnh cách hiển thị dữ liệu:

Chọn dữ liệu phân tích

  • Chọn cột tỉnh/thành: Từ dropdown, chọn cột chứa tên tỉnh/thành phố (ví dụ: "Ten_Tinh_Thanh")
  • Chọn loại dữ liệu: Chọn loại dữ liệu cần phân tích (ví dụ: "Vi phạm giao thông")
  • Chọn cột dữ liệu: Chọn cột cụ thể chứa dữ liệu bạn muốn phân tích (ví dụ: "MAVP")

Tùy chọn hiển thị

Đánh dấu vào các ô để chọn cách hiển thị kết quả:

  • Hiển thị bản đồ: Hiển thị dữ liệu trên bản đồ Việt Nam
  • Hiển thị biểu đồ cột: Hiển thị dữ liệu dưới dạng biểu đồ cột
  • Hiển thị bảng dữ liệu: Hiển thị dữ liệu dưới dạng bảng chi tiết

Bảng màu bản đồ

Chọn bảng màu cho bản đồ từ dropdown (mặc định: "Đỏ")

  • Các tùy chọn bao gồm: Đỏ, Xanh dương, Xanh lá, Vàng-Cam-Đỏ, Tím, Đỏ-Xanh, Viridis, Plasma

Xử lý dữ liệu

Sau khi hoàn tất các tùy chọn, nhấn nút "Xử lý dữ liệu" để phân tích và hiển thị kết quả.

Xem kết quả phân tích

Sau khi xử lý dữ liệu, bạn sẽ thấy các phần kết quả phân tích:

Thống kê tổng quan

Hiển thị các chỉ số quan trọng:

  • Tổng số: Tổng số dữ liệu đã chọn trên toàn quốc
  • Trung bình mỗi tỉnh: Giá trị trung bình trên mỗi tỉnh/thành
  • Cao nhất: Tỉnh/thành có giá trị cao nhất và giá trị tương ứng
  • Thấp nhất: Tỉnh/thành có giá trị thấp nhất và giá trị tương ứng

Thống kê tổng hợp

Hiển thị tổng hợp theo các loại dữ liệu khác nhau:

  • Tổng số vi phạm
  • Tổng số tai nạn
  • Tổng số tử vong
  • Tổng số bị thương
  • Tổng mức phạt

Bản đồ Choropleth

  • Bản đồ màu thể hiện mức độ giá trị trên các tỉnh/thành
  • Di chuột qua mỗi tỉnh/thành để xem chi tiết
  • Màu sắc càng đậm thể hiện giá trị càng cao

Biểu đồ cột

Hiển thị top 10 tỉnh/thành có giá trị cao nhất theo loại dữ liệu đã chọn.

Bảng dữ liệu

  • Hiển thị chi tiết dữ liệu của tất cả tỉnh/thành, được sắp xếp theo thứ tự giảm dần
  • Có thể sắp xếp và lọc dữ liệu bằng cách nhấn vào tiêu đề cột hoặc sử dụng ô tìm kiếm

Thiết lập ngôn ngữ

VNTraffic hỗ trợ nhiều ngôn ngữ, có thể chọn từ góc trên bên phải của giao diện:

  • Tiếng Việt (mặc định)
  • English (Tiếng Anh)
  • 日本語 (Tiếng Nhật)
  • Русский (Tiếng Nga)

Quản lý tài khoản người dùng

Tùy chọn tài khoản nằm ở góc trên bên phải (Xin chào [Tên người dùng]):

  • Hồ sơ: Xem và chỉnh sửa thông tin cá nhân
  • Báo cáo: Xem các báo cáo đã tạo
  • Yêu cầu dữ liệu: Yêu cầu dữ liệu bổ sung
  • Cài đặt: Thay đổi cài đặt tài khoản
  • Đăng xuất: Đăng xuất khỏi hệ thống

Cấu trúc dữ liệu đầu vào

Yêu cầu file GeoJSON

File GeoJSON cần có cấu trúc:

{
  "type": "FeatureCollection",
  "features": [
    {
      "type": "Feature",
      "properties": {
        "ten_tinh": "Tên tỉnh/thành phố"
      },
      "geometry": {
        // Dữ liệu hình học của tỉnh/thành
      }
    }
    // Các tỉnh/thành khác
  ]
}

Yêu cầu file CSV

File CSV nên có các cột sau (có thể thêm các cột khác):

Ten_Tinh_Thanh,MAVP,MATN,TUVONG,BITHUONG,Tổng mức phạt
Hà Nội,123,45,5,20,40000000
TP. Hồ Chí Minh,234,56,8,25,50000000
...

Trong đó:

  • Ten_Tinh_Thanh: Tên tỉnh/thành phố
  • MAVP: Mã vi phạm hoặc số lượng vi phạm
  • MATN: Mã tai nạn hoặc số lượng tai nạn
  • TUVONG: Số người tử vong
  • BITHUONG: Số người bị thương
  • Tổng mức phạt: Tổng số tiền phạt (đơn vị: VNĐ)

Lưu ý quan trọng: Đảm bảo tên tỉnh/thành trong file CSV khớp chính xác với thuộc tính ten_tinh trong file GeoJSON để dữ liệu được hiển thị chính xác trên bản đồ.

Xử lý lỗi thường gặp

Một số lỗi phổ biến và cách khắc phục:

  1. Lỗi file không đúng định dạng: Đảm bảo bạn đang tải lên đúng loại file (GeoJSON và CSV).
  2. Lỗi không tìm thấy cột: Kiểm tra xem bạn đã chọn đúng cột trong file CSV.
  3. Lỗi không hiển thị bản đồ: Có thể do file GeoJSON có vấn đề về tọa độ. Ứng dụng sẽ cố gắng tự động sửa lỗi, nhưng trong một số trường hợp, bạn có thể cần sửa file GeoJSON trước khi tải lên.
  4. Không khớp dữ liệu: Đảm bảo tên tỉnh/thành trong file CSV khớp chính xác với thuộc tính ten_tinh trong file GeoJSON.
  5. Lỗi khi chuyển đổi ngôn ngữ: Nếu giao diện không cập nhật khi chuyển đổi ngôn ngữ, hãy thử làm mới trang.
